	Seventh lab for Data Structures and Algorithms, in which we made a heap data 
structure, specifically a min-heap. The heap is capable of inserting, deleting, 
resizing, and heapifying non-heap arrays, among other things. To run the unit tests,
run Test.java from the Lab7 folder.
	
	Files included:
		- README.txt (this)
		- UR_Heap (the provided class I implemented)
		- Heap.java (the heap implementation)
		- Test.java (my test class)
