Greetings, traveler, and welcome to this README file.
To build the project, unzip the folder and navigate (through the command prompt or IDE terminal) to the correct folder.
Then, enter: gcc -o Project4 -std=c99 -Wall -Werror -pedantic Project4.c
Finally, enter .\Project4 to run the compiled program.

Note: when inputting unspecified attributes, I used "*" for strings and -1 for integers. Additionally, when printing a table, the program prints the hash table as-is, including any indices that contain NULL.
