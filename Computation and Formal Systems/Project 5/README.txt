Greetings, traveler, and welcome to this README file.
To build the project, unzip the folder and navigate (through the command prompt or IDE terminal) to the correct folder.
Then, enter: gcc -o project5 -std=c99 -Wall -Werror strdup.c Boolean.c Gates.c Circuit.c project5.c
Finally, enter .\project5 to run the compiled program.

Note: to avoid warnings for unused functions, I commented out some code in Circuit.c.
