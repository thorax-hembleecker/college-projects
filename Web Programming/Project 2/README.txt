This is a website for sharing stories!

Navigation is in the top navbar, and will update with new options depending on if you are logged in.
Tables are User, Story, and Chapter. They're pretty self-explanatory. User can be inserted to by signing up, and the other two can do both insertion and deletion.

All important URLs can be found by navigating throughout the site.

Additional requirements:
- User authentication
- Insertion/deletion on Story and Chapter ("New Story", "Delete Chapter"/"Delete Story")
