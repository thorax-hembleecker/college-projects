Greetings, traveler, and welcome to this README file.
To build the project, unzip the folder and navigate (through the command prompt or IDE terminal) to the correct folder.
Then, enter "javac State.java" to build the program, and "java State" to run it.
(My terrible need/habit for compact project folders led me to write every class and method inside the State file. One day I will know better than to do this, but today is not that day.)

Note: to make a move in Pawntastic, simply type the name of the square of the pawn you want to move, followed by a space, then the name of the square you want to pawn to move to (e.g. "c2 c4", "a4 a5"). Additionally, you can simply press ENTER without typing anything to end the game automatically.
