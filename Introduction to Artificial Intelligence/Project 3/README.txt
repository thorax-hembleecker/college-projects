Greetings, traveler, and welcome to this README file.
To build the project, unzip the folder and navigate (through the command prompt or IDE terminal) to the correct folder.
Then, enter "javac Inferinator.java Test.java bn\base\*.java bn\core\*.java bn\parser\*.java" to build the program, and "java Test bn\examples\[file name] [query variable] [evidence variable] [value]" to run the program on a given file. If there are multiple evidence variables, you can input them immediately after the others, with a space in between: i.e. "java Test bn\examples\aima-alarm.xml B J true M true".
